Web Landing Page Mockup

Introduction

This is a web design mockup of an e-commerce camera landing page designed in 3 hours. The top navigation consists of important links for common product categories as well as a search bar and cart. When hovered over each of the links will expand into a mega menu to help the user navigate. Following the navigation is a timed banner to create a sense of urgency. The large hero portion of the wesite features an engaging full width image, a compelling headline (and subheadline), and large CTA button. Under the hero are some feature products organized in material-style cards and a footer with important links as well as ways to follow the brand.
