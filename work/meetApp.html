MeetApp is a solution for the difficulty of organizing events. It allows you to create events with your friends and family and track where everyone is on event day to ensure everyone is arriving on time and safely! The motivation for this app came from the desire to know where students are during large off-campus school-hosted events. The app is also great for ensuring everyone gets home safe after late nights of parting. Some of the core features of the app include being able to create events and select locations using MapKit, sending notifications for event reminders, and being able to track friends' locations during the day of the event.


Features:

Register and Login pathways that connect to Firebase authentication
Home screen for viewing events
Event creation screen
Map view for searching locations
Event details screen
Group Map to see your friends' and family's locations on event day
